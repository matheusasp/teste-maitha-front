This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.1.4.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.


## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.



Este é o frontend do projeto. Antes de prosseguir, verifique se todos os campos "apiUrl" nos arquivos relevantes estão configurados corretamente para apontar para o projeto do backend.

A configuração correta do campo "apiUrl" é crucial para que o frontend se comunique corretamente com o backend e acesse os dados e funcionalidades necessários.

Certifique-se de que o campo "apiUrl" esteja configurado corretamente nos arquivos relevantes, como configurações de ambiente, arquivos de serviço ou qualquer outro lugar onde a URL da API seja definida.

Se você estiver enfrentando problemas de comunicação entre o frontend e o backend, verifique se a URL do backend está corretamente configurada no campo "apiUrl". Isso garantirá que as solicitações de API sejam direcionadas ao local correto.
